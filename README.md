Smart Material Selection

Smart Material Selection is an AI-powered web application designed to help users identify and explore suitable smart materials for engineering and modern technological applications.

📌 About The Project

This project focuses on simplifying the process of selecting smart materials based on their properties, behavior, and engineering applications. It provides an interactive and modern web interface for students, researchers, and engineering enthusiasts.

Smart materials are materials that can respond to environmental changes such as:

- Temperature
- Pressure
- Magnetic fields
- Electric fields
- Stress or strain

Examples include:

- Shape Memory Alloys (SMA)
- Piezoelectric Materials
- Magnetostrictive Materials
- Electroactive Polymers
